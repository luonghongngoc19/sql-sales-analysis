# SQL Sales Analysis Project | AdventureWorks

## Overview

This project analyzes sales data from the AdventureWorks database using SQL Server to extract business insights, calculate key performance indicators (KPIs), and perform customer segmentation.

The goal is to support data-driven decision-making by identifying revenue trends, customer behavior, and operational performance.

---

## Tools Used

- SQL Server
- AdventureWorks Database
- SQL (JOIN, CTE, Window Functions, Aggregation)

---

## Business Questions

This project answers key business questions, including:

- How has revenue changed over time?
- What are the top-performing products?
- Who are the highest-value customers?
- Which territories generate the most revenue?
- What is the repeat purchase rate?
- How effective are marketing campaigns?

---

## Key KPIs Calculated

- Revenue by Year
- Year-over-Year (YoY) Growth
- Average Order Value (AOV)
- Revenue per Employee
- Repeat Purchase Rate
- On-time Delivery Rate
- Discount Revenue Share

---

## Advanced Analysis

### Customer Segmentation using RFM Model

Customers were segmented based on:

- Recency
- Frequency
- Monetary value

Customer groups identified:

- Best Customers
- Big Spenders
- Loyal Customers
- At-Risk Customers
- Lost Customers

This helps businesses improve customer retention and marketing strategies.

---

## Key Insights

- Revenue showed strong growth from 2011 to 2013
- Growth was driven mainly by increased order volume
- Average Order Value declined, indicating smaller basket sizes
- Repeat purchase rate was approximately 39%
- Revenue was concentrated in specific products and territories
- Marketing campaigns had varying levels of effectiveness

---

## Business Recommendations

- Increase Average Order Value through bundles and upselling
- Focus on high-value customer segments
- Optimize marketing campaigns based on performance
- Improve customer retention through loyalty programs
- Optimize inventory management to reduce dead stock
